# ECG Arrhythmia Classification using MIT-BIH Dataset

Project Overview
This project implements an end-to-end deep learning pipeline for automated heartbeat classification using the MIT-BIH Arrhythmia Dataset. The system performs ECG signal preprocessing, heartbeat segmentation around R-peaks, patient-level data splitting to prevent leakage, class imbalance handling using weighted sampling, and training of a neural network model for multi-class arrhythmia detection.
